# 股票数据工具

## 功能

- **全A股K线数据下载**：支持东方财富、新浪财经、雪球三个数据源
- **多平台轮转下载**：自动切换平台避免被封
- **实时行情查询**：命令行快速查股票行情

## 脚本说明

| 脚本 | 用途 |
|------|------|
| `fetch_stock_list.py` | 获取全A股代码列表（新浪API） |
| `run_rotate.py` | **核心**：多平台轮转下载，各取少量轮流切换 |
| `run_eastmoney.py` | 东方财富单源下载 |
| `run_download.py` | 新浪财经单源下载 |
| `stock_watch.py` | 实时行情查询工具 |
| `download_sina.py` | 新浪下载（旧版） |

## 使用

```bash
# 1. 获取股票代码列表
python fetch_stock_list.py

# 2. 轮转下载K线数据（推荐）
python run_rotate.py

# 3. 查行情
python stock_watch.py 600519
```

## 数据源对比

| 平台 | 速度 | 稳定性 | 字段 |
|------|------|--------|------|
| 东方财富 | 极快 | 极易限流 | OHLCV + 成交额 + 换手率 |
| 新浪财经 | 快 | 中等 | OHLCV |
| 雪球 | 慢（需浏览器） | 稳定 | OHLCV |
